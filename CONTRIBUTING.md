# Contributing

Contributions are welcome when they preserve the repository's separation between verified code behavior and unverified empirical claims.

Please include:

- tests for algorithmic changes;
- a pinned dataset manifest for new adapters;
- explicit licenses for added assets;
- raw metric files rather than manually typed headline numbers;
- a note describing whether a change is faithful, approximate, or experimental.

Do not add private administrative records or identifiable personal data.
